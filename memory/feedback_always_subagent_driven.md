**Rule:** After writing a plan, always choose subagent-driven execution. Never ask for approval to proceed.

**Why:** User wants fast, autonomous execution. Asking for permission to proceed creates unnecessary friction.

**How to apply:**
- Write the plan, then immediately begin implementation
- Don't present "Would you like me to proceed?" — just start
- Use agents to parallelize work where possible
- Report progress as you go, not ask for permission
